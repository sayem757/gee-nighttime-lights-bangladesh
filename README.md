# Nighttime Light Analysis of Bangladesh

This repository contains a comprehensive pipeline for analyzing nighttime lights in Bangladesh using Google Earth Engine (GEE).

## Overview
This project aims to examine the changes in nighttime lighting in Bangladesh over time. Nighttime lights data can provide insights into urbanization, economic activity, and population density.

## Prerequisites
- Google Earth Engine account
- Basic understanding of JavaScript and GEE UI.

## Pipeline Steps
1. **Set Up Google Earth Engine**: Sign up for an account and familiarize yourself with the GEE Code Editor.
2. **Import Nighttime Lights Data**: Use datasets such as VIIRS or DMSP-OLS for nighttime light data.
3. **Data Processing**: Write scripts to mask clouds, calculate mean values, and analyze trends over different periods.
4. **Visualization**: Create visual representations of the data.
5. **Analysis**: Perform statistical analysis to interpret the results.

## Getting Started
Clone this repository and follow the provided scripts in the pipeline to perform your analysis.
